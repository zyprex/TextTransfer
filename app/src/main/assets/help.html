TextTransfer

QUICK HELP

  1. Let all devices connected to same Wi-Fi network.
  2. Use the switch to turn on http server.
  3. Check the app's notification. Open Webpage link address which is showing in notification bar.
  4. Click button line below start type. After finish typing, click Submit
  5. On any other device's web browser, click Refresh, the text should appeared.

User Guide In Detail

Uri Class

GET method

  • / text transfer page
  • /r/ return raw text
  • /f/ list of file in root directory
  • /f?p=/sdcard list of file in directory 'sdcard'
  • /u/ upload file page

POST method

  • /?text=FindMe synchronize text "FindMe" to http server
  • /upload/ upload result page

Text Transfer Interface

This is the main page

  • Submit: Synchronize text to http server
  • Refresh: Get text from http server
  • Files: View file list on server's root directory
    • Can live view small file, such as text file and image file.
    • Can download file via http server.
  • Upload: Upload file to http server, and save it in 'Download' folder. Max file size is 1025 MiB.
